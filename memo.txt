以下の手順で進めてください。

1. JavaプログラムのJARファイルを作成する
まず、JavaプログラムをコンパイルしてJARファイルを作成します。以下のコマンドをターミナルで実行します。

sh
コードをコピーする
javac -d out src/com/yourpackage/*.java
jar cvf yourprogram.jar -C out .
2. シェルスクリプトを作成する
次に、シェルスクリプトを作成してJARファイルを実行するようにします。テキストエディタで新しいファイルを作成し、以下の内容を入力します。

sh
コードをコピーする
#!/bin/bash

# Javaのパスが環境変数に含まれていない場合は、適宜設定してください
export JAVA_HOME=/path/to/your/java/home
export PATH=$JAVA_HOME/bin:$PATH

# JARファイルのディレクトリに移動
cd /path/to/your/jar

# JARファイルを実行
java -jar yourprogram.jar
このスクリプトをrun.shという名前で保存します。

3. シェルスクリプトに実行権限を付与する
保存したシェルスクリプトに実行権限を付与します。以下のコマンドをターミナルで実行します。

sh
コードをコピーする
chmod +x /path/to/your/script/run.sh
4. シェルスクリプトを実行する
シェルスクリプトを実行してJavaプログラムを起動します。以下のコマンドをターミナルで実行します。

sh
コードをコピーする
/path/to/your/script/run.sh
5. 自動実行の設定（オプション）
特定の時間に自動で実行したい場合は、cronジョブを設定します。以下のコマンドをターミナルで実行してcronの設定を編集します。

sh
コードをコピーする
crontab -e
以下の行を追加して、毎日午前9時にシェルスクリプトを実行する設定を行います。

sh
コードをコピーする
0 9 * * * /path/to/your/script/run.sh
これで、指定した時間に自動的にスクリプトが実行され、Javaプログラムが起動します。

これらの手順で、Mac上でJavaプログラムを自動で実行することができます。何か不明な点や追加の質問があれば教えてください。
